Prompt: Pengembangan Modul Kurikulum LMS DonatJS
Peran: Bertindaklah sebagai Instructional Designer dan Senior Web Developer yang ahli dalam menyusun kurikulum teknis berbasis proyek.

Tugas: Buatkan data JSON untuk variabel learn di dataset.js untuk kursus "Fullstack Web Development with DonatJS". Modul ini harus dibagi menjadi kategori-kategori logis (misalnya: Fondasi, UI/UX, Backend & Logic, Final Project).

Spesifikasi Struktur Data:

Gunakan format: learn: { categories: [ { name: 'Nama Kategori', items: [ ... ] } ] }.

Setiap item (modul) harus memiliki: id, title, dan lines (array of strings).

Panduan Konten (Pedagogi):

Narasi: Gunakan bahasa yang profesional namun memotivasi.

Komponen Visual (Wajib Digunakan): Gunakan sintaks khusus DonatJS di dalam array lines:

### Judul: Untuk sub-bab.

---: Untuk garis pemisah.

skill:[persentase]:[Judul]:[Keterangan]: Untuk target capaian (contoh: skill:90%:DOM Mastery:Manipulasi elemen tanpa jQuery).

card:[Judul]=[Subjudul]:[Deskripsi]: Untuk menyoroti konsep kunci.

step:[Tahun/Urutan]:[Judul]:[Deskripsi]: Untuk alur kerja atau roadmap.

table:[{"Kolom1": "Isi", "Kolom2": "Isi"}]: Untuk instruksi tugas atau jadwal praktikum (format JSON string di dalam array).

Daftar Modul yang Harus Dibuat:

Modul 1 (Fondasi): Dasar JavaScript Modern (ES6+), variabel, dan arrow functions.

Modul 2 (Arsitektur): Pengenalan MVC di DonatJS, State Management, dan Routing.

Modul 3 (UI Komponen): Pembuatan Modal, Toast Notification, dan Layout dinamis (gunakan contoh Pertemuan 4 yang sudah ada sebagai referensi gaya bahasa).

Modul 4 (Interaksi Data): Menghubungkan Form ke Dataset dan simulasi penyimpanan data.

Aturan Penulisan:

Pastikan ID modul unik (contoh: js-basics, mvc-arch).

Jangan gunakan library eksternal dalam penjelasan materi.

Fokus pada Clean Code dan Minimalism.

Cara Menggunakan Prompt Ini:
Gunakan prompt ini setiap kali Anda ingin menambah kategori baru atau memperdalam isi modul.

Jika Anda ingin fokus pada satu modul saja (misal hanya Modul Database), cukup ubah bagian Daftar Modul yang Harus Dibuat menjadi spesifik.

Hasil dari AI bisa langsung Anda copy-paste ke dalam objek learn di file dataset.js.

Dengan prompt ini, AI akan otomatis memberikan output yang mengandung tabel tugas, bar persentase kemampuan (skill bar), dan kartu-kartu penjelasan yang membuat tampilan LMS Anda terlihat sangat profesional.
